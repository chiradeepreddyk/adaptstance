About AdaptStance

What is AdaptStance?

AdaptStance is a browser-based biomechanical analysis tool for adaptive-stance motorcycles that can shift between Cruiser and Sport riding postures by actuating the handlebar, seat, and/or footpeg positions.

The Rider Triangle

The three key hardpoints — handlebar grip, seat (hip), and footpeg — form the rider triangle. All joint positions are derived from these three points together with the rider's body segment lengths.

Biomechanical Model

A 2-D sagittal-plane (side-view) model is used. The lower body (hip → knee → ankle) and upper body (shoulder → elbow → wrist) are each solved as two-link inverse-kinematics (IK) chains. Torso lean angle is estimated by finding the back angle at which the arms achieve approximately 75 % of maximum reach — a natural, non-fatiguing posture.

Joints Modelled

Wrist · Elbow · Shoulder · Back (torso lean) · Knee · Ankle. Each joint has a configurable target range for Cruiser and Sport modes. The default ranges are informed by SAE J1516, ISO 15534-1, and published motorcycle ergonomics research.

Repository Structure

index.html — standalone web application (this file)
src/biomechanics.js — IK solver and angle calculations
src/renderer.js — SVG stick-figure renderer
src/ui.js — tab navigation, form helpers, results display
data/defaults.json — default body measurements and angle ranges
tests/ — unit tests (Node.js / Jest)
